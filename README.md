# Avaliação para Desenvolvedor Full Stack

Erik Atilio Silva Rey

##### Desafio:
1. Criar uma aplicação web que seja responsável pelo cadastro de colaboradores com
as seguintes informações: **cpf, nome, cargo, setor.**  

2. Além do cadastro de novos colaboradores, a aplicação deve atender aos
requisitos:
    - Listar os colaboradores por setor, listar os colaboradores por cargo;
    - Editar as informações cadastradas sobre colaboradores;
    - Deletar colaboradores;
    - Utilizar um token de sessão que devera expirar em **30 minutos** sendo necessário renova-lo;
    - Modelar o banco de dados e encaminhar o diagrama Entidade Relacionamento;
    - Layout e design a sua escolha;

***Obs:*** *Por se tratar de um CRUD acredito que o ideal seria utilizar uma lib de banco para evitar códigos SQL, mas fica por decisão do candidato.*  

***Para o front-end utilizar Angular.**  
***Para o backend utilizar Node js.**  
***Utilizar banco de dados MySQL ou Postgres**  

Devido a utilização do NodeJS informar se está utilizando o **YARN** ou o **NPM** e
disponibilizar junto o arquivo ***package.json.***
